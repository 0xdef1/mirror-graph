# Mirror Graph

## Prerequisites
* Node.js v12
* TypeScript v3.8
* GraphQL v15
* yarn (https://yarnpkg.com/)
* PostgreSQL v12 (https://www.postgresql.org/download/)
* ormconfig.json on project root
* default.js on /config

## Install dependencies
```
$ yarn
```

## Run Collector
```
$ yarn collect
```

## Run Graph Server
```
$ yarn start
```
